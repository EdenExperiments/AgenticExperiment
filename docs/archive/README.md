# Documentation Archive Policy

This directory stores superseded operational/process documentation that is no longer canonical.

## Archive Rules

- Every archived file should include:
  - an archive date,
  - the reason it was superseded,
  - a pointer to the replacement canonical document.
- Do not edit archived files for active process guidance.
- Active guidance must live in `AGENTS.md`, `Documentation/README.md`, or `docs/CURSOR-AGENT-HANDBOOK.md`.
